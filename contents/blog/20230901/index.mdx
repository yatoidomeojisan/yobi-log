# 民訴

<Kihan title="当事者の意義とその確定基準">
当事者とは、訴えまたは訴えられることにより、判決の名宛人となるべきものを言う。当事者が誰かは人的裁判籍等の判断基準となるので訴えの提起後直ちに、かつ明確に判断する必要があるところ、訴えの提起後においては、訴状の記載がもっと明確と言えるため、訴状の記載を基準とすべきである。もっとも、具体的妥当性にも考慮する必要があるので、当事者は、訴状の当事者欄の記載のみならず、訴状全体の記載から判断すべきと考える。
</Kihan>

氏名冒用訴訟における再審について、338条1項3号の事由は、絶対的上告事由の場合と同様、氏名冒用訴訟など、当事者に対する実質的手続保障が欠ける場合も含むと解される。
