# 行政法

## 公法私法二元論

公法関係と私法関係では適用される法原理が異なるとする見解があるが、もはや行政上の法律関係を、公法関係と私法関係に分ける実益は無い。そこで公法関係と私法関係を区別せず、当該処分の性質・実体法の趣旨・問題状況を考慮して、私法規定の適用の有無を決すべきである。
