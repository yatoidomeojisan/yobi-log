# 刑法

<Kihan title="過失">
過失とは構成要件的要素であり、予見可能性を前提とする予見義務違反、及び結果回避可能性を前提とする結果回避義務違反からなるものと理解すべきである。
</Kihan>

<Kihan title="因果関係">
因果関係は偶然発生した結果を行為に帰責させないようにすうるものであるから、行為の危険が結果へと現実化したと言える場合に、因果関係は認められると考える。そして、行為後に介在事情が存在する場合は、その判断にあたり、行為の重大性、介在事情の寄与度・異常性を考慮する。
</Kihan>

# 刑訴法

<Kihan title="「強制の処分」（197条1項ただし書き）">
科学技術が捜査に応用されること及び刑事訴訟法における強制処分に関する要件・手続きの厳格性に鑑み、「強制の処分」とは、処分者の明示または黙示の意思に反して、重要な権利利益を制約する処分を言うと考える。
</Kihan>

<Kihan title="職務質問に伴う有形力の行使">
警職法2条1項が非侵害的行政行為しか許容していなければ同項の存在意義はなく、同項は一定の有形力行使の根拠規定となりうる。また、犯罪の予防鎮圧と言う行政警察活動目的達成のために、一定の有形力の行使を認める必要がある。したがって、同項に基づき、有形力の行使自体は可能と考える。もっとも、強制にわたる行為を禁止した警職法2条3項より、 強制捜査手続きによらなければ許されないような強制処分に至る有形力の行使は許されない。また、強制に至らなくても、警察比例の原則(警職法1条2項参照)より、必要性・緊急性を考慮し、具体的状況のもとで相当と認められる限度で適法になると考える。
</Kihan>

<Kihan title="承諾なき所持品検査の可否">
警職法2条1項は職務質問について規定するのみであるが、所持品検査は口頭による質問と密接に関連し、かつ職務質問の効果をあげるうえで必要性・有効性の認められる行為であるから、職務質問に付随してこれを行うことができると考える。もっとも、所持品検査は任意手段たる職務質問の付随行為である以上、原則として相手方の承諾が必要である。ただし、犯罪の予防・鎮圧と言う行政警察活動目的の達成のため、承諾がない場合であっても、捜索に至らない程度の行為は強制にわたらない限り、所持品検査の必要性・緊急性、これにより侵害される個人の法益と保護されるべき公益との均衡を考慮し、 具体的状況のもとで相当と言える限度においてのみ許容されると考える。
</Kihan>

<Kihan title="自動車検問">
自動車検問は、相手方の人の協力を求める形で行われ、自動車利用者の自由を不当に制約することにならない方法、態様で行われる場合には適法である。
</Kihan>
