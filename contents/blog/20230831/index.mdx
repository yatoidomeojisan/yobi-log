# 刑法

<Kihan title="不作為犯の実行行為性">
実行行為とは構成要件的結果発生の現実的危険性を有する行為を言うところ、不作為によってもかかる危険を惹起できる。もっとも、不作為犯は作為犯よりも自由が制約される度合いが大きく、処罰範囲を限定する必要もある。そこで、作為との構成要件的同価値性が認められる場合に、不作為犯の実行行為性が認められると考える。具体的には、①作為義務、②作為の可能性・容易性が必要と考える。
</Kihan>

# 刑訴法

<Kihan title="告訴を欠く親告罪捜査の可否">
親告罪における告訴は訴訟条件であるとされており、捜査の条件では無いから、捜査機関が犯罪ありと思料した場合（189条2項）には告訴がなくても捜査が可能であると言える。もっとも、告訴が得られる可能性が全くない場合には、公訴提起の余地がないので捜査の必要性は認められず、捜査は許されないと解する。
</Kihan>

<Kihan title="告訴の主観的不可分と相対的親告罪">
共犯者の1人または数人に対して行われた告訴は、他の共犯者に対しても効力を生じる（238条1項）。もっとも、相対的親告罪の場合、主観的不可分の原則が妥当しない。なぜなら、相対的親告罪は、犯人と被害者の特殊な人的関係に配慮して定められた規定であるが、親族でない共犯者に対する告訴が親族である犯人にも及ぶとなれば、かかる規定の趣旨が没却されてしまうからである。
</Kihan>

<Kihan title="告訴の客観的不可分">
明文はないが、告訴は、犯罪事実に対して行うものであるから、原則として、単一の犯罪の一部についてした告訴はその全部に対して効力を生じる。 もっとも観念的競合の犯罪事実で、被害者が複数の場合には、一方の被害者が告訴をしても、他方の被害者が告訴したことにはならない。被害者意思を尊重すべきだからである。 また、非親告罪に限定した告訴については、親告罪に及ばない。親告罪とした趣旨を没却してしまうからである。
</Kihan>

# 民法

<Kihan title="「目的の範囲内」（34条）の意義">
法人は社会的に有用な一定目的のために権利義務の主体たる地位を認められたものであるし、また、文言上も「権利を有し、義務を負う」とされていることから、「目的」によって権利能力が制限されるものと解する。
</Kihan>

<Kihan title="「目的の範囲内」（34条）の判断方法">
（意義の続き）目的の範囲外の行為は絶対的無効とされるから、取引の安全を害する程度が大きい。したがって、当該行為が「目的の範囲内」にあるか否かを判断するにあたっては、柔軟に解釈する必要がある。具体的には、「目的の範囲」には目的を遂行するのに間接的に必要な行為まで広く含み、また、目的遂行に必要か否かは行為の客観的な性質に即し抽象的に判断すべきである。（なお、非営利法人の場合には、設立の根拠法との関係上、より厳格に解する必要がある。）
</Kihan>

<Kihan title="「目的の範囲」（34条）外の行為の効力">
（意義の続き）そうすると、目的の範囲外の行為は絶対的に無効である。ただし、事情によっては、このような目的制限による無効を主張することが信義則（1条2項）に反する場合がある。
</Kihan>

# 商法

<Kihan title="商法504条ただし書きと消滅時効">
両債権は相手方の選択以前には併存していると解されるが、一方が選択されればもう一方はその主張ができなくなると言う関係において単に権利の帰属者の点においてのみ択一的な債権として並存しているに過ぎず、債権の実体は単一であると解する。 このような債権の実態的単一性ないし相関的性格に鑑みれば、本人の請求は訴訟継続中、代理人の債権につき催告（民法150条1項）に準じた時効完成猶予の効力を及ぼすと解するのが相当である。
</Kihan>

<Kihan title="商人資格の取得時期">
商事取引における取引の安全を図るため、基準の明確性が重要となる。そこで、事業意思の客観的認識可能性が生じた時点で「商人」となると解すべきである。もっとも行為者が営業のために行った準備行為として取引関係に入った相手方は商法の適用があると考えるのが通常だから、「商人」として、商法の適用を肯定して構わないと考える。
</Kihan>
